## Descripción:

El proyecto consiste en un gestor de tareas. La realización del proyecto se ha separado para Alumno A y Alumno B.

1. Alumno A:
    - Menu
    - Visualización de tareas

    - Eliminar una tarea

## Instalación y configuración

### 1. Tecnologías necesarias
* **Python 3.x** instalado.
* **Git** instalado.

### 2. Clonación del repositorio
```bash
git clone [URL_DEL_REPOSITORIO]
cd [NOMBRE_DE_LA_CARPETA]
```
### 3. Crear y activar el entorno vitual

1. En Linux/macOS:<br><br>
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. En Windows:<br><br>
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
### 4. Instalar dependencias
```bash
    pip install -r requirements.txt
```
